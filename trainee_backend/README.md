# trainee_backend
This Repo Contains Task mentioned in LMS Version Control.
This Readme is for Backend.


1. Install Dependencies
- Make sure you have Node.js installed.
- npm install


2. Run the Project
- npm start
- node index.js

3. Open in Browser
- http://localhost:3000

4. Testing
- Verify API calls (if frontend is connected)
- Check browser console for errors
- Use Postman for API Testing.
